# IBM Applied Data Science Capstone Project – SpaceX Falcon 9

## Project Overview
This project analyzes SpaceX Falcon 9 launch data to predict first stage landing success.

## Tasks Completed
- Data collection using SpaceX API
- Web scraping from Wikipedia
- Data wrangling and cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Machine Learning prediction

## Files
- jupyter-labs-spacex-data-collection-api-v2.ipynb → API data collection
- jupyter-labs-webscraping.ipynb → Web scraping
- (More notebooks will be added)
